# テキサスホールデム ポーカーゲーム

テキサスホールデムポーカーゲームです。2つのバージョンがあります：
- **Web版**: ブラウザで遊べるグラフィカル版
- **Python版**: コマンドラインで遊べるテキスト版

## 特徴

- 完全なテキサスホールデムのルール実装
- 1人プレイヤー vs 3人のAI
- ベッティングシステム（チェック、コール、レイズ、フォールド）
- 自動役判定（ロイヤルフラッシュ～ハイカードまで全10種類）
- ブラインド制度
- ショーダウン機能

## Web版（推奨）

### 実行方法

1. `index.html` をブラウザで開く

```bash
# macOSの場合
open index.html

# Windowsの場合
start index.html

# Linuxの場合
xdg-open index.html
```

または、ファイルマネージャーから `index.html` をダブルクリックしてください。

### 遊び方

1. ブラウザでゲームが起動します
2. プレイヤー名と初期チップを設定
3. 「ゲーム開始」ボタンをクリック
4. 画面上のボタンでアクション（フォールド、チェック、コール、レイズ）を選択
5. ゲームの進行状況とログが画面に表示されます

## Python版

### 必要要件

### 実行方法

```bash
cd /Users/dw1003/poker-game

```

## 遊び方

1. ゲームを起動すると、名前の入力を求められます
2. 各プレイヤーに初期チップ1000が配られます
3. 各ハンドは以下の流れで進行します：
   - ブラインド（強制ベット）
   - プリフロップ（ホールカード2枚配布後のベッティング）
   - フロップ（コミュニティカード3枚公開後のベッティング）
   - ターン（コミュニティカード4枚目公開後のベッティング）
   - リバー（コミュニティカード5枚目公開後のベッティング）
   - ショーダウン（役の比較）

## ベッティングアクション

- **[c]heck**: ベットせずに次のプレイヤーに回す（ベットがない場合のみ）
- **[c]all**: 現在のベット額に合わせる
- **[r]aise**: ベット額を上げる
- **[f]old**: ハンドを降りる

## 役の強さ（強い順）

1. ロイヤルフラッシュ - 同じスートのA-K-Q-J-10
2. ストレートフラッシュ - 同じスートの連続する5枚
3. フォーカード - 同じランクのカード4枚
4. フルハウス - スリーカード + ワンペア
5. フラッシュ - 同じスートのカード5枚
6. ストレート - 連続する5枚のカード
7. スリーカード - 同じランクのカード3枚
8. ツーペア - ペア2組
9. ワンペア - 同じランクのカード2枚
10. ハイカード - 上記のいずれにも該当しない

## ゲームの設定

デフォルトの設定：
- 初期チップ: 1000
- スモールブラインド: 10
- ビッグブラインド: 20
- プレイヤー数: 4人（人間1人 + AI 3人）

設定を変更したい場合は、`main.py`の以下の部分を編集してください：

```python
players = [
    HumanPlayer(player_name, chips=1000),  # 初期チップを変更
    AIPlayer("AI_1", chips=1000, aggression=0.3),  # AIの攻撃性を調整（0.0～1.0）
    # プレイヤーを追加/削除可能
]

game = TexasHoldem(players, small_blind=10, big_blind=20)  # ブラインド額を変更
```
